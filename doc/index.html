渦輪の形成と伝播の3次元シミュレーション + 可視化

Smoke Ring

概要

3次元ナビエ・ストークス方程式を解いて渦輪の形成と伝播のシミュレーションを行う。

シミュレーション領域

直方体領域。3次元周期境界条件。カーテシアン座標。

計算手法

空間離散化は2次中心差分法。時間積分は4次ルンゲ・クッタ法。

実行方法

  • cd src
  • make
  • cd ../job
  • qsub sample.js
